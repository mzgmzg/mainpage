# 极简浏览器主页

一个极简风格的浏览器起始页，支持自定义书签、多搜索引擎切换、必应每日壁纸、一言、天气等功能。

## 功能

- **时钟** — 24 小时制数字时钟，中文日期，按时段问候
- **搜索** — 支持 Google / Bing / 百度，图标化切换
- **书签** — 可拖拽排序的常用网站网格，支持添加/编辑/删除
- **背景** — 多图片源切换（必应每日、随机风景、纯色渐变、自定义 URL）
- **天气** — 基于浏览器定位，降级默认北京
- **每日一言** — 一言 API，中文句子
- **亮色/暗色** — 手动切换或跟随系统
- **配置管理** — 导入/导出 JSON 配置文件，恢复默认
- **Docker 部署** — 一键构建部署

## 快速开始

```bash
# 开发（热更新）
make dev

# 生产部署
make up

# 停止
make down
```

访问 `http://localhost:8080`。

## 配置

所有配置存储在浏览器 localStorage 中：
- `hp_settings` — 搜索引擎、主题、背景源等
- `hp_bookmarks` — 书签列表

点击右上角齿轮进入设置面板，可管理书签、切换背景源、导入导出配置。

## 技术栈

Vue 3 + Vite + Docker + Nginx
